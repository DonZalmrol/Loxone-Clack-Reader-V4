# Customized-Clack-Reader-V4
ESPHome firmware for Clack DV water softener monitoring via M5Stack Atom S3 Lite. Tracks salt level (VL53L1X TOF), water usage, 3-channel power (INA3221), and regeneration cycles. Serves a JSON API and HTML5 dashboard for Loxone Miniserver integration — no Home Assistant required.
